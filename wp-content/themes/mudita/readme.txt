== Description ==

Mudita is a  beautifully designed, clean and modern multi-purpose WordPress theme. The theme is very versatile and can be used by any business websites, digital agency, consultancy, corporate business, freelancers, portfolio and product showcase, bloggers and everyone one who appreciates beautiful design. The theme is SEO friendly, which  make it easy for your site to rank Google and other search engines. Mudita is loaded with several features to make user-friendly, interactive and visually stunning website. Such features include two menus, full width banner,  service section, team section, testimonial section,  off canvas menu and social media. It has four footer area and a right sidebar and includes  three custom widgets for recent posts, popular posts, and the featured post. The theme is rigorously tested and optimized for speed and faster page load time and has a secure and clean code. The theme is also translation ready. Designed with visitor engagement in mind, Mudita allows anyone to create a stunning, fast and mobile friendly website compatible with all browsers and devices. Check the demo at http://raratheme.com/preview/mudita/, Documentation at http://raratheme.com/documentation/mudita and get support at http://raratheme.com/support-forum/.


== Installation ==
        
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Documentation ==
1. Setting up the Homepage
    * To set up the homepage you must first create a new page. 
    * Create a new Page
    * Go to Pages > Add New.
    * Enter the Page Title for the Page
    * You do not require to enter anything in the content section for creating a home page. 
    * Select the Homepage Template from the Page Attributes section 
 
2. Configuring the front Page display setting 
    * Go to Appearance>Customize > Default Settings> Static Front Page. 
    * Select A static page under Front Page displays
    * Choose the page you just created as Front page.
    * Click Save & Publish.

3. Header Settings 
    * Go to Appearance>Customize>Header Settings.
    * Check Enable Header Info 
    * Enter your details on Header Phone.
    * Enter Save & Publish.

4. Banner Section
    * Go to Appearance>Customizer>Home Page Settings>Banner Section
    * Check Enable Banner Section
    * Select Banner Post. The feature Image of the selected Post will Appear as a Banner Image.
    * Enter Read More Text.
    * Click Save & Publish.
    
5. Featured Section
    * Go to Appearance>Customizer>Home Page Settings>Featured Section
    * Check Enable Featured Section.
    * Enter Featured Section Title and Featured Section Content. 
    * Select The Featured Post One, Featured Post Two, Featured Post Three and Featured Post Four. The Feature Image of the selected posts will be displayed under the section in the Homepage.
    * Enter Read More Text.
    * Click Save & Publish. 

6. Service Section 
    * Go to Appearance>Customizer>Home Page Settings>Service Section
    * Check Enable Service Section.
    * Select Service Post One and Service Post Two. The feature Image of the selected posts will appear under this section.
    * If the selected post don't have featured image the post will not be displayed in Service section. So, the featured image must be added to the respective posts.    
    * Enter Read More Text. 
    * Click Save & Publish. 
    
7. Team Section
    * Go to Appearance>Customizer>Home Page Settings>Team Section
    * Check Enable Team Section.
    * Enter Team Section Title and Team Section Content.
    * Select Categoryfor Team Section. The Content and the feature image of the posts under the selected category will be displayed under the Team Section
    * Click Save & Publish.

8. Blog Section
    * Go to Appearance>Customizer>Home Page Settings>BlogSection
    * Check Enable Blog Section.
    * Enter Blog Section Title
    * Check Enable Blog Date to display the blog date.
    * Click Save & Publish. 

9. Testimonial Section
    * Go to Appearance>Customizer>Home Page Settings>Testimonial Section
    * Check Enable Testimonial Section.
    * Enter Testimonial Section Title and Testimonial Section Content.
    * Select Testimonial Post One, Testimonial Post Two and Testimonial Post Three
    * Click Save & Publish. 

10. Breadcrumb Settings
    * Go to Appearance>Customizer>Breadcrumb Settings
    * Check Enable Breadcrumb
    * Check Show Current, if you wish to display the current Page on the breadcrumb.
    * Enter Breadcrumb Home Text
    * Enter Breadcrumb Separator
    * Click Save & Publish.

11. Social Settings
    * Go to Appearance>Customizer>Social Settings
    * Check Enable Social Section
    * Enter Social Section title and link of your social Media Pages.
    * Click Save & Publish.

12. Custom CSS Settings
    * Go to Appearance >Customize>Custom CSS settings.
    * Enter the required custom CSS. For example, to change the body font size to 17 Pixels: body{font-size: 17px}
    * Click Save & Publish.
